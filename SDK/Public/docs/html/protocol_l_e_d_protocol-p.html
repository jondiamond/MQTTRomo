Detailed Description

Protocol that provides access to the robot's RMCoreLEDs.

This protocol simply gives access to a read-only instance of RMCoreLEDs, the direct interface to controlling the LEDs on an RMCoreRobot. Note that although this instance is read-only, its members are accessed directly to control the LED.
