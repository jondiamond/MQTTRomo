Detailed Description

An RMCoreRobotRomo3 is a subclass of RMCoreRobot that defines which protocols are supported by ROMO3A.

ROMO3A conforms to DifferentialDriveProtocol (which inherits from DriveProtocol), HeadTiltProtocol, and LEDProtocol.
