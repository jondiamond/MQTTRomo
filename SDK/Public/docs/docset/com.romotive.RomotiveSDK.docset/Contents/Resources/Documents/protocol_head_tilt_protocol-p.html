Detailed Description

The protocol for directly interfacing with the up/down tilting degree of freedom that controls the docked iDevice.

This protocol provides methods for interacting with a robot containing a tilting axis. When an RMCoreRobot object is instantiated (and when it implements the HeadTiltProtocol), you may send the robot commands and query the state of its head tilt through the functionality in this protocol.
