Detailed Description

An RMCoreRobot object is a generic representation of a robot. It contains the robot's vitals and properties about the robot (whether it's connected, what type of commands it will respond to). A specific robot implementation will be a subclass of RMCoreRobot (e.g., RMCoreRobotRomo3).
